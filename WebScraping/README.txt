Script en Python que hace Web Scraping a una pagina web

Para usar el script unicamente es necesario cambiar la parte donde dice que ingreses la liga de la pagina a la cual deseas hacer el Web Scraping 
Por fines ilustrativos se tiene de manera predeterminada la pagina de la UANL

Hará las instrucciones necesarias y al final mostrará en pantalla la información obtenida

Al final guardará un archivo .txt con la información (Formato html, se pueden hacer ciertos cambios para unicamente tomar la información deseada)
